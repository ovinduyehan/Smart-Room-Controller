Developed a Smart Room Controller using Arduino Uno, a PIR motion sensor, and a DHT22 temperature and humidity sensor in Wokwi simulation. The system automatically controls room lighting based on occupancy and operates a fan based on temperature conditions, demonstrating basic home automation principles.

Features

Motion-based lighting control
Temperature-based fan control
Real-time environmental monitoring
Automated decision-making using sensor inputs
Serial Monitor status reporting

Skills Learned

Multi-sensor integration
Arduino Programming (C++)
Home automation logic
Conditional decision-making
Environmental monitoring
Input/Output control
Embedded systems development

Technologies Used

Arduino Uno
PIR Motion Sensor
DHT22 Sensor
LEDs (Light & Fan Indicators)
Wokwi Simulator
Learning Outcome

This project introduced the concept of combining multiple sensors and outputs within a single system. It helped develop skills in automation logic, system design, and real-time decision-making, forming the foundation for future smart home and IoT applications.
